AIMA-CPP

A C++ library for implementation of machine learning algorithms provided by Stuart Russell and Peter Norvig in AI: A Modern Approach, 3rd Edition. Currently, decision tree learning is being implemented. Future plans include the entirety of supervised learning, unsupervised learning, and neural networks.

LineFunk

LineFunk is a simple and easy to run tool that allows students in Linear Algebra to compute and check their works. The software currently populates matrices, depopulate matrices, create empty matrices, etc.

BeesKnees

BeesKnees is an app using Google Maps API to locate the nearest bathroom. Should the user need to find a bathroom quickly, they will be able to find public bathrooms that are available to the public, not bathrooms that are only available to the facility's tenants/customers. Not only that, the user will be able to rate and review bathrooms. There is also the ability for organizational users to promote their facilities by promoting the quality of their bathrooms.
